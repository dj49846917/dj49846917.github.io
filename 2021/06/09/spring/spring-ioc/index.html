Springз®Җд»Ӣ

SpringжҳҜд»Җд№Ҳ

  • SpringжҳҜеҲҶеұӮзҡ„ Java SE/EEеә”з”Ё full-stack иҪ»йҮҸзә§ејҖжәҗжЎҶжһ¶пјҢд»Ҙ IoCпјҲInverse Of ControlпјҡеҸҚиҪ¬жҺ§еҲ¶пјүе’Ң AOPпјҲAspect Oriented Programmingпјҡйқўеҗ‘еҲҮйқўзј–зЁӢпјүдёәеҶ…ж ёгҖӮ
  • жҸҗдҫӣдәҶеұ•зҺ°еұӮ SpringMVC е’ҢжҢҒд№…еұӮ Spring JDBCTemplate д»ҘеҸҠдёҡеҠЎеұӮдәӢеҠЎз®ЎзҗҶзӯүдј—еӨҡзҡ„дјҒдёҡзә§еә”з”ЁжҠҖжңҜпјҢиҝҳиғҪж•ҙеҗҲејҖжәҗдё–з•Ңдј—еӨҡи‘—еҗҚзҡ„з¬¬дёүж–№жЎҶжһ¶е’Ңзұ»еә“пјҢйҖҗжёҗжҲҗдёәдҪҝз”ЁжңҖеӨҡзҡ„Java EE дјҒдёҡеә”з”ЁејҖжәҗжЎҶжһ¶гҖӮ

SpringеҸ‘еұ•еҺҶзЁӢ

Springзҡ„дјҳеҠҝ

  1. ж–№дҫҝи§ЈиҖҰпјҢз®ҖеҢ–ејҖеҸ‘
    • йҖҡиҝҮ Spring жҸҗдҫӣзҡ„ IoCе®№еҷЁпјҢеҸҜд»Ҙе°ҶеҜ№иұЎй—ҙзҡ„дҫқиө–е…ізі»дәӨз”ұ Spring иҝӣиЎҢжҺ§еҲ¶пјҢйҒҝе…ҚзЎ¬зј–з ҒжүҖйҖ жҲҗзҡ„иҝҮеәҰиҖҰеҗҲгҖӮ
      з”ЁжҲ·д№ҹдёҚеҝ…еҶҚдёәеҚ•дҫӢжЁЎејҸзұ»гҖҒеұһжҖ§ж–Үд»¶и§ЈжһҗзӯүиҝҷдәӣеҫҲеә•еұӮзҡ„йңҖжұӮзј–еҶҷд»Јз ҒпјҢеҸҜд»Ҙжӣҙдё“жіЁдәҺдёҠеұӮзҡ„еә”з”ЁгҖӮ
  2. AOPзј–зЁӢзҡ„ж”ҜжҢҒ
    • йҖҡиҝҮ Springзҡ„ AOP еҠҹиғҪпјҢж–№дҫҝиҝӣиЎҢйқўеҗ‘еҲҮйқўзј–зЁӢпјҢи®ёеӨҡдёҚе®№жҳ“з”Ёдј з»ҹ OOP е®һзҺ°зҡ„еҠҹиғҪеҸҜд»ҘйҖҡиҝҮ AOP иҪ»жқҫе®һзҺ°гҖӮ
  3. еЈ°жҳҺејҸдәӢеҠЎзҡ„ж”ҜжҢҒ
    • еҸҜд»Ҙе°ҶжҲ‘д»¬д»ҺеҚ•и°ғзғҰй—·зҡ„дәӢеҠЎз®ЎзҗҶд»Јз Ғдёӯи§Ји„ұеҮәжқҘпјҢйҖҡиҝҮеЈ°жҳҺејҸж–№ејҸзҒөжҙ»зҡ„иҝӣиЎҢдәӢеҠЎз®ЎзҗҶпјҢжҸҗй«ҳејҖеҸ‘ж•ҲзҺҮе’ҢиҙЁйҮҸгҖӮ
  4. ж–№дҫҝзЁӢеәҸзҡ„жөӢиҜ•
    • еҸҜд»Ҙз”Ёйқһе®№еҷЁдҫқиө–зҡ„зј–зЁӢж–№ејҸиҝӣиЎҢеҮ д№ҺжүҖжңүзҡ„жөӢиҜ•е·ҘдҪңпјҢжөӢиҜ•дёҚеҶҚжҳҜжҳӮиҙөзҡ„ж“ҚдҪңпјҢиҖҢжҳҜйҡҸжүӢеҸҜеҒҡзҡ„дәӢжғ…гҖӮ
  5. ж–№дҫҝйӣҶжҲҗеҗ„з§Қдјҳз§ҖжЎҶжһ¶
    • ж–№дҫҝйӣҶжҲҗеҗ„з§Қдјҳз§ҖжЎҶжһ¶
  6. йҷҚдҪҺ JavaEE API зҡ„дҪҝз”ЁйҡҫеәҰ

  7. Javaжәҗз ҒжҳҜз»Ҹе…ёеӯҰд№ иҢғдҫӢ

  4. еңЁdaoдёӢж–°е»әimpl.UserDaoImplзҡ„е®һзҺ°зұ»пјҢеҺ»е®һзҺ°UserDaoзҡ„saveж–№жі•, е№¶ж·»еҠ @RepositoryжіЁи§Ј
  5. еңЁjavaдёӢж–°е»әcom.itheima.service.UserServiceзҡ„жҺҘеҸЈ
  6. еңЁserviceдёӢж–°е»әimpl.UserServiceImplзҡ„е®һзҺ°зұ»пјҢеҺ»е®һзҺ°saveResultж–№жі•пјҢе№¶ж·»еҠ @ServiceпјҢ@ResourceжіЁи§Ј
  7. еңЁapplicationContext.xmlдёӯе°ҶиҝҷдәӣжіЁи§ЈйҖҡзҹҘspringпјҢйңҖж·»еҠ contextж Үзӯҫ(еңЁbeansж·»еҠ xmlns:contextе’Ңxsi:schemaLocation)
  8. зј–еҶҷжөӢиҜ•зұ»Test

@Component

дҪҝз”ЁеңЁзұ»дёҠпјҢз”ЁдәҺе®һдҫӢеҢ–bean

  • еңЁдёҠйқўдёҚеҗ«жіЁи§Јзҡ„д»Јз Ғзј–еҶҷзҡ„з¬¬еӣӣжӯҘпјҢж·»еҠ @ComponentжіЁи§Ј

  • жӯҘйӘӨ5зҡ„й…ҚзҪ®еҸҜд»ҘеҲ жҺү

@Service

дҪҝз”ЁеңЁServiceеұӮзұ»дёҠпјҢз”ЁдәҺе®һдҫӢеҢ–bean

  • еңЁдёҠйқўдёҚеҗ«жіЁи§Јзҡ„д»Јз Ғзј–еҶҷзҡ„з¬¬7жӯҘпјҢж·»еҠ @ServiceжіЁи§Ј

@Repository

дҪҝз”ЁеңЁdaoеұӮзұ»дёҠпјҢз”ЁдәҺе®һдҫӢеҢ–bean

  • еңЁдёҠйқўдёҚеҗ«жіЁи§Јзҡ„д»Јз Ғзј–еҶҷзҡ„з¬¬еӣӣжӯҘпјҢж·»еҠ @RepositoryжіЁи§Ј

  • жӯҘйӘӨ5зҡ„й…ҚзҪ®еҸҜд»ҘеҲ жҺү

@Autowiredе’Ң@Qualifier

@Autowiredпјҡз”ЁдәҺж №жҚ®зұ»еһӢдҫқиө–жіЁе…Ҙ
@Qualifierпјҡз»“еҗҲ@AutowiredдёҖиө·дҪҝз”ЁпјҢз”ЁдәҺж №жҚ®еҗҚз§°дҫқиө–жіЁе…Ҙ

  • еңЁдёҠйқўдёҚеҗ«жіЁи§Јзҡ„д»Јз Ғзј–еҶҷзҡ„з¬¬7жӯҘпјҢж·»еҠ @AutowiredгҖҒ@QualifierжіЁи§Ј

@Resource

зӣёеҪ“дәҺ@Autowired+@QualifierпјҢжҢүз…§еҗҚз§°иҝӣиЎҢжіЁе…Ҙ

  • еңЁдёҠйқўдёҚеҗ«жіЁи§Јзҡ„д»Јз Ғзј–еҶҷзҡ„з¬¬7жӯҘпјҢж·»еҠ @AutowiredгҖҒ@QualifierжіЁи§Ј

@Value

жіЁе…Ҙжҷ®йҖҡеұһжҖ§

  • еңЁдёҠйқўдёҚеҗ«жіЁи§Јзҡ„д»Јз Ғзј–еҶҷзҡ„з¬¬еӣӣжӯҘпјҢж·»еҠ nameе’ҢageдёӨдёӘжҲҗе‘ҳеҸҳйҮҸпјҢдҪҝз”Ё@ValueжіЁи§Ј

  • жӯҘйӘӨ5зҡ„й…ҚзҪ®еҸҜд»ҘеҲ жҺү

  • жөӢиҜ•д»Јз Ғ2пјҡз»“еҗҲdruidдҪҝз”Ё@ValueеҸҳйҮҸ

    1. ж–°е»әitheima_spring_di3зҡ„module,е№¶й…ҚзҪ®webappж–Үд»¶еӨ№
    2. еңЁpom.xmlж·»еҠ druidгҖҒjunitгҖҒspring-contextгҖҒmysqlеҢ…
    3. еңЁresourcesдёӢж–°е»әjdbc.properties
    4. еңЁresourcesдёӢж–°е»әapplicationContext.xmlпјҢе№¶еј•е…Ҙjdbc.properties
    5. еңЁjavaдёӢж–°е»әcom.itheima.dao.StudentDao
    6. еңЁdaoдёӢж–°е»әimpl.StudentDaoImplзҡ„е®һзҺ°зұ»пјҢдҪҝз”Ё@RepositoryгҖҒ@ValueжіЁи§Ј
    7. еңЁjavaдёӢж–°е»әcom.itheima.service.StudentServiceзҡ„дёҡеҠЎеұӮ
    8. еңЁserviceдёӢж–°е»әStudentServiceImplзҡ„е®һзҺ°зұ»пјҢдҪҝз”Ё@Serviceе’Ң@ResourceжіЁи§Ј
    9. зј–еҶҷжөӢиҜ•ж–№жі•

Springж–°жіЁи§Ј

  • дҪҝз”ЁдёҠйқўзҡ„жіЁи§ЈиҝҳдёҚиғҪе…ЁйғЁжӣҝд»Јxmlй…ҚзҪ®ж–Үд»¶пјҢиҝҳйңҖиҰҒдҪҝз”ЁжіЁи§Јжӣҝд»Јзҡ„й…ҚзҪ®еҰӮдёӢпјҡ
    • йқһиҮӘе®ҡд№үзҡ„Beanзҡ„й…ҚзҪ®пјҡ<bean>
    • еҠ иҪҪpropertiesж–Үд»¶зҡ„й…ҚзҪ®пјҡ<context:property-placeholder>
    • з»„д»¶жү«жҸҸзҡ„й…ҚзҪ®пјҡ<context:component-scan>
    • еј•е…Ҙе…¶д»–ж–Үд»¶пјҡ<import>

дёҚеҗ«ж–°жіЁи§Јзҡ„д»Јз Ғзј–еҶҷ

  1. ж–°е»әitheima_spring_di4зҡ„module,е№¶й…ҚзҪ®webappж–Үд»¶еӨ№
  2. еңЁpom.xmlдёӯеј•е…Ҙspring-contextгҖҒmysql-connector-javaгҖҒdruidеҢ…
  3. еңЁresourcesдёӢж–°е»әjdbc.propertiesж–Үд»¶еӨ№
  4. еңЁresourcesдёӢж–°е»әapplicationContext.xmlпјҢеј•е…Ҙjdbc.properties
  5. еңЁjavaдёӢж–°е»әcom.itheima.dao.StudentDao
  6. еңЁdaoдёӢж–°е»әimpl.StudentDaoImplзұ»пјҢе№¶дҪҝз”Ё@RepositoryгҖҒ@ValueжіЁи§Ј
  7. еңЁjavaдёӢж–°е»әcom.itheima.service.StudentServiceзҡ„дёҡеҠЎеұӮ
  8. еңЁapplicationContext.xmlдёӯж·»еҠ жү«жҸҸжіЁи§Ј
  9. зј–еҶҷжөӢиҜ•ж–№жі•

дҪҝз”Ёж–°жіЁи§Јд»Јз Ғзј–еҶҷ

  1. еңЁеүҚйқўдёҚеҗ«ж–°жіЁи§Јзҡ„д»Јз Ғз¬¬5жӯҘеүҚпјҢеңЁjavaдёӢж–°е»әcom.itheima.Config.SpringConfigй…ҚзҪ®зұ»пјҢе№¶еҶҷе…Ҙ

  2. applicationContext.xmlе°ұе®Ңе…ЁжІЎжңүз”ЁдәҶ

  3. зј–еҶҷжөӢиҜ•ж–№жі•

@Configuration

з”ЁдәҺжҢҮе®ҡеҪ“еүҚзұ»жҳҜдёҖдёӘ Spring й…ҚзҪ®зұ»пјҢеҪ“еҲӣе»әе®№еҷЁж—¶дјҡд»ҺиҜҘзұ»дёҠеҠ иҪҪжіЁи§Ј

  • еңЁдёҠйқўж·»еҠ ж–°жіЁи§Јзҡ„д»Јз ҒдёӯпјҢ@ConfigurationжіЁи§ЈпјҢзӣёеҪ“дәҺе‘ҠиҜүspringиҝҷжҳҜж ёеҝғй…ҚзҪ®зұ»

@ComponentScan

з”ЁдәҺжҢҮе®ҡ Spring еңЁеҲқе§ӢеҢ–е®№еҷЁж—¶иҰҒжү«жҸҸзҡ„еҢ…гҖӮдҪңз”Ёе’ҢеңЁ Spring зҡ„ xml й…ҚзҪ®ж–Үд»¶дёӯзҡ„<context:component-scan base-package="com.itheima"/>дёҖж ·

  • еңЁдёҠйқўж·»еҠ ж–°жіЁи§Јзҡ„д»Јз ҒдёӯпјҢ@ComponentScanжіЁи§ЈпјҢжӣҝжҚўдәҶapplicationContext.xmlдёӯзҡ„<context:component-scan base-package="com.itheima"/>

@PropertySource

з”ЁдәҺеҠ иҪҪ.properties ж–Үд»¶дёӯзҡ„й…ҚзҪ®

  • еңЁдёҠйқўж·»еҠ ж–°жіЁи§Јзҡ„д»Јз ҒдёӯпјҢ@PropertySourceжіЁи§ЈпјҢжӣҝжҚўдәҶapplicationContext.xmlдёӯзҡ„<context:property-placeholder location="jdbc.properties"></context:property-placeholder>

@Bean

дҪҝз”ЁеңЁж–№жі•дёҠпјҢж ҮжіЁе°ҶиҜҘж–№жі•зҡ„иҝ”еӣһеҖјеӯҳеӮЁеҲ° Spring е®№еҷЁдёӯ

  • еңЁдёҠйқўж·»еҠ ж–°жіЁи§Јзҡ„д»Јз ҒдёӯпјҢ@BeanжіЁи§ЈпјҢеҠ дёҠдёҖдёӘйҡҸдҫҝе®ҡд№үзҡ„getDataSourceеҮҪж•°пјҢжӣҝжҚўдәҶapplicationContext.xmlдёӯзҡ„

@Import

з”ЁдәҺеҜје…Ҙе…¶д»–й…ҚзҪ®зұ»

  • еңЁдёҠйқўж·»еҠ ж–°жіЁи§Јзҡ„д»Јз ҒдёӯпјҢSpringConfigдёӯзҡ„й…ҚзҪ®еҫҲеӨҡзҡ„ж—¶еҖҷпјҢеҸҜд»ҘиҝӣиЎҢжӢҶеҲҶпјҢеңЁjavaдёӢж–°е»әcom.itheima.Config.DataSourceConfigзҡ„й…ҚзҪ®зұ»пјҢжҠҠжүҖжңүи·ҹж“ҚдҪңж•°жҚ®еә“зӣёе…ізҡ„й…ҚзҪ®ж”ҫеҲ°йҮҢйқў

  • еңЁSpringConfigй…ҚзҪ®зұ»дёӯдҪҝз”Ё@Importеј•е…Ҙ
